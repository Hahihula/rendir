# Welcome

This is a multilingual blog demonstrating Rustpress internationalization (i18n) support.

## Features

- Automatic language detection from directory structure
- Translation fallback when content is missing
- Language selector component
- SEO-friendly hreflang tags

## Try It

Edit any markdown file and refresh - the dev server will rebuild automatically!
