# Erste Schritte mit Rust

Rust ist eine Systembrogrammiersprache, die auf Sicherheit, Nebenläufigkeit und Leistung ausgerichtet ist.

## Warum Rust?

- Speichersicherheit ohne Garbage Collection
- Null-Overhead-Abstraktionen
- Furchtlose Nebenläufigkeit
- Hervorragende Werkzeuge (Cargo, rustfmt)

## Erste Schritte

1. Installieren Sie Rust über rustup
2. Erstellen Sie Ihr erstes Projekt mit `cargo new`
3. Schreiben Sie Code!
