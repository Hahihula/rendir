# Introduction

Welcome to the Example Book! This book demonstrates rustpress's mdbook ingestion capabilities.

## What is mdbook?

mdbook is a popular Rust tool for creating books from Markdown files. It powers the [Rust Book](https://doc.rust-lang.org/book/) and many other documentation projects.

## Why rustpress?

Rustpress can ingest existing mdbook projects and render them with custom templates, enabling you to:

- Customize the visual design while preserving the structure
- Add new components (alerts, tabs, etc.) to your existing documentation
- Integrate with your own build pipeline
